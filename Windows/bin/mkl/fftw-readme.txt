FFTW Interface to Intel® Math Kernel Library - 11.0

Scilab 5.x uses fftw implentation of Intel Math Kernel Library for fast fourier transformation.

Intel® Math Kernel Library (Intel® MKL) offers FFTW3 interfaces to Intel MKL 
Fast Fourier Transform and Trigonometric Transform functionality.
The purpose of these interfaces is to enable applications using FFTW (www.fftw.org)
to gain performance with Intel MKL without changing the program source code.

FFTW3 interfaces are provided in open source as FFTW wrappers to Intel MKL.
For ease of use, FFTW3 interface is also integrated in Intel MKL.


Processors Minimum Recommended Supported :

* Intel® Xeon® processor family     
* Intel® Core™ processor family     
* Intel® Pentium® 4 processor family      
* AMD Athlon processor      
* AMD Opteron processor
